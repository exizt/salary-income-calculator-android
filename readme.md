# 버저닝 룰

버전 구분 : `{Major} {Minor} {Build}`
* Major : 큰 변경이 있을 시 증가.
* Minor : 어느 정도 유의미한 변경이 있을 시 증가.
* Build : 무조건적으로 증가. versionCode 와 동일하거나 그보다 높게 설정.
